# Predicción de Episodios de Mala Calidad del Aire en la Ciudad de México

## Descripción

Este proyecto tiene como objetivo desarrollar modelos de aprendizaje automático para predecir episodios de mala calidad del aire utilizando datos de PM2.5 y variables meteorológicas obtenidas del Sistema Nacional de Información de la Calidad del Aire (SINAICA).

## Variables Utilizadas

- PM2.5
- Temperatura
- Humedad Relativa
- Velocidad del Viento
- Presión Atmosférica

## Modelos Implementados

1. Regresión Logística
2. Random Forest
3. Gradient Boosting
4. Red Neuronal Artificial

## Resultados

| Modelo | Accuracy |
|----------|----------|
| Regresión Logística | 82.95 % |
| Random Forest | 86.36 % |
| Gradient Boosting | 87.50 % |
| Red Neuronal | 80.68 % |

El mejor modelo fue Gradient Boosting con una precisión de 87.50 %.
